导入 ArcPy

安装 ArcGIS 的时候一般会附带安装 Python，但是其版本比较旧，例如 ArcGIS 10.1 自带的还是 Python 2.6，而我通常选择使用较新的 Python 2.x 版本，这就造成了在执行跟 ArcPy 相关的命令时，将提示 No module named arcpy 的问题。对此，ArcGIS 官方文档提供了一个导入 ArcPy 的方法。
