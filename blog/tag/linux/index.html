周五 31 八月 2012

安装

我下载的镜像为 Fedora-17-x86_64-Live-KDE.iso，利用 Universal USB Installer 软件进行U盘安装。

中文支持

周一 19 十二月 2011

折腾 Ubuntu 时出现了不少问题，凭印象整理了几个小问题，备忘一下。环境：Ubuntu 11.10 64-bit

安装 Ubuntu 时卡在「正在探测文件系统」

安装前打开终端，硬盘安装输入 sudo umount -l /isodevice

光盘安装输入 sudo umount -l /cdrom

依次运行下面的命令：

sudo apt-get update

sudo apt-get dist-upgrade

sudo apt-get install –reinstall software-center
